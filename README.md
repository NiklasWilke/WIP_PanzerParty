## Anleitung
1. NodeJS-Server starten -> **run.bat** ausführen
2. **http://&lt;*ip-adresse*&gt;:8000** im Browser aufrufen
3. Map auswählen
4. Spieler via QR-Code oder IP verbinden
5. Party!
